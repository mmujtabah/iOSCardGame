# Card Game (iOS)

A simple SwiftUI card game where you and the CPU draw random cards.
The higher card wins the round and gets one point.

## Features

- Random card dealing for Player and CPU
- Live score tracking
- Clean SwiftUI interface with custom assets

## Run Locally

1. Open `Card Game.xcodeproj` in Xcode.
2. Select an iOS Simulator.
3. Build and run the app.
